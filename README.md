🤿 SubmarineSite: Deep Diving into Angular & TypeScript

Welcome to SubmarineSite! This project is a personal deep dive (pun intended) into modern web development. It’s a study project where I’ve focused on rebuilding a diving niche landing page with a minimalist, high-end aesthetic that’s as clear as tropical waters.

🌊 Why this project?

I built this to master Angular 17 and TypeScript while experimenting with sleek design patterns. My goal was to create a site that doesn't just look "modern," but also feels incredibly robust under the hood—something that would make a professional dive shop owner say, "I need this."

🛠 The "Gear" (Tech Stack)

    Framework: Angular 17.3.17 (The engine)

    Language: TypeScript (For strictly typed, clean, and reliable code)

    Design Philosophy: Modern Minimalism (Focus on whitespace, bold typography, and immersive imagery)

    CLI: Angular CLI (For efficient scaffolding)

🚀 Getting Started
Prerequisites

Make sure you have Node.js and the Angular CLI installed.
Installation & Launch

    Clone the repo: git clone [repo-link]

    Install dependencies: npm install

    Launch the project: ```bash
    ng serve

    Navigate: Open http://localhost:4200/. The app will live-reload as you make changes to the code.

🏗 Development Workflow

    Components & Logic: I used TypeScript to ensure every component and service has a clear structure. To add more, use: ng generate component component-name.

    Production Build: Run ng build to store the optimized artifacts in the dist/ directory.

    Safety Checks: Run ng test to execute unit tests via Karma.

📉 Personal Study Notes

    TypeScript Mastery: Leveraging interfaces and types to handle diving gear data and site configurations without runtime errors.

    Angular Signals/Control Flow: Using the latest Angular features to keep the UI snappy.

    The "Human" Touch: Moving away from generic templates to create a unique, brand-focused experience for the diving community.
